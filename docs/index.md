# Dokumentacja podejścia Ds / TPE

Ta dokumentacja opisuje model myślenia o relacjach **Ds**, **TPE**, 24/7 oraz **consensual slavery** jako o długoterminowej, odpowiedzialnej strukturze relacyjnej, a nie jako o zbiorze scen, technik czy doświadczeń erotycznych.

Punktem wyjścia jest założenie, że władza w relacji nie jest dekoracją, rolą ani narzędziem intensywności. Jest zobowiązaniem. Istnieje wyłącznie tam, gdzie została świadomie oddana, może zostać realnie wycofana i jest prowadzona przez osobę zdolną ponosić konsekwencje swoich decyzji.

## Cel dokumentacji

Celem tego miejsca jest uporządkowanie zasad, pojęć, granic i standardów dotyczących:

- relacji opartych na głębokiej wymianie władzy,
- odpowiedzialności osoby dominującej,
- dobrowolności i możliwości wycofania zgody,
- stabilności długoterminowych kontraktów,
- funkcjonowania społeczności Ds / TPE,
- rozróżnienia między autonomią relacji a nadużyciem.

Nie jest to instrukcja „jak dominować”. Nie jest to podręcznik scen. Nie jest to zaproszenie do kolekcjonowania doświadczeń.

To próba opisania systemu, w którym relacja, władza, odpowiedzialność i wspólnota tworzą spójną całość.

## Założenia podstawowe

Relacja **Ds** / **TPE** w tym ujęciu jest traktowana jako poważna, długoterminowa struktura życia. Może obejmować głębokie podporządkowanie, daleko idącą kontrolę i trwałe zobowiązania, ale jej fundamentem zawsze pozostaje dobrowolność.

Osoba dominująca nie otrzymuje władzy „z definicji”. Otrzymuje ją dlatego, że druga osoba świadomie ją przekazuje. Z tego wynika obowiązek prowadzenia, ochrony stabilności relacji, pilnowania granic oraz reagowania na skutki własnych decyzji.

Osoba uległa nie jest traktowana jako rekwizyt relacji ani dowód statusu dominującego. Jej oddanie ma wartość tylko wtedy, gdy jest świadome, dobrowolne i osadzone w realnym zaufaniu.
